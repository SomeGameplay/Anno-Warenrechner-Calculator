# Anno-Warenrechner-Calculator
Dieser Warenrechner zeigt dir auf Basis deiner Einwohnerzahlen direkt den benötigten Warenbedarf, passende Produktionsketten und wie gut deine aktuelle Versorgung bereits abgedeckt ist. Vorhandene Gebäude lassen sich bequem eintragen, Spielstände speichern und per Overlay sogar direkt während des Spielens ein- und ausblenden.
